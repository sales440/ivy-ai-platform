Our Training Programs

BASIC CNC SYSTEM TRAINING

This course is designed to help users understand the basic machine operator process
What You'll Learn:
  • Learning navigation of the user interface menus
  • Performing system backups and restore
  • Parameters (Meanings, adjustments, modification)
  • PLC Programming
  • Machine monitoring
  • Logic analyzer
  • Editing & Definitions
  • Saving & Compiling
  • Creating, testing, and verifying simple G-code programs
  • Conversational programming
Who Should Attend:

Machine operators, maintenance technicians, production supervisors, anyone who interacts with FAGOR CNC systems daily.

After This Course:

You'll operate your CNC system confidently, troubleshoot basic issues independently, and understand the fundamentals that make advanced techniques possible.

ADVANCE CNC SYSTEM TRAINING

Tailored for high level end users, CNC integrators, and OEMs to learn advanced system capabilities
What You'll Master:
  • Parameters for drives - adjusting and saving
  • Loading parameters from CNC and PC
  • Software changes - Upgrade / downgrade software
  • Different drives integration: BCSD, AXD, Quercus
  • Using FGUIM (built in software) for screen customizations / modifications
  • Introduction for high level language instructions to control the flow of the programs
Who Should Attend:

Advanced users, CNC integrators, OEMs, technical specialists, anyone responsible for system configuration, customization, or integration.

After This Course:

You'll configure and customize FAGOR systems at expert level, integrate different drive systems, modify screen interfaces, and control program flow with advanced instructions.
